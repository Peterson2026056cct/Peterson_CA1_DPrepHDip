## Description
This project focuses on preparing a student dataset for analysis. It includes data cleaning, transformation, visualization, and dimensionality reduction using PCA.

## Problem
The dataset contains inconsistencies such as missing values, incorrect formats, and mixed data types, which make analysis difficult.

## Solution
The data was cleaned by handling missing values, correcting errors, and converting data types. Visualization and correlation analysis were used to better understand the dataset, followed by PCA for dimensionality reduction.

## Objectives
lean and prepare the dataset  
Understand data distribution and patterns  
Identify relationships between variables  
Apply PCA to reduce dimensionality  
Prepare data for future modeling
